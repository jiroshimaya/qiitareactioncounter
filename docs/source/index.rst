qiitareactioncounter documentation
==================================


バージョン: |version|

Qiitaの記事に対するリアクション数を集計・分析するツールです。

主な機能
--------

- Qiitaの記事に対するリアクション数の取得
- リアクション数の集計と分析
- CSV形式での結果出力
- 指定した期間における指定したユーザとQiita全体のリアクション数の取得、比較、分析
